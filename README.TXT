# Timelines Of Triumph

Game edukasi sejarah Indonesia buatan saya menggunakan Greenfoot.

🎮 Fitur:
- Soal sejarah acak per level
- Player menyerang jika jawab benar, monster menyerang jika salah
- Sistem darah (HP), poin, bintang, dan artefak
- Toko artefak + sistem penyimpanan
- Transisi antar level dan animasi

📌 Dibuat menggunakan:
- Greenfoot
- Bahasa Java
- Gambar & suara pendukung

📂 Cara menjalankan:
1. Buka Greenfoot
2. Klik "Open Project" dan pilih folder ini
3. Jalankan game dengan tombol "Run"
